Cheat Sheet pour Stimulus.js

Introduction à Stimulus.js

Stimulus est une bibliothèque JavaScript légère qui facilite le développement d’interactions basées sur des données dans vos applications web. Elle est conçue pour fonctionner en harmonie avec les frameworks JavaScript existants et se concentre sur l’amélioration progressive de vos pages web.

Principes Clés

0. Créer un contrôleurs

Documentation controlleurs Stimulus

Pour créer un contrôleur, je créé un nouveau fichier dans mon dossier controllers nommé exemple_controller.js

javascript/application.js

import { Application } from "stimulus";
import ExampleController from "./controllers/example_controller"; // Importez votre contrôleur

// Initialise l'application Stimulus
const application = Application.start();

// Charger individuellement les contrôleurs
application.register("example", ExampleController);

1. Contrôleurs (Controllers)

Documentation controlleurs Stimulus

Les contrôleurs sont les blocs de construction de Stimulus. Chaque contrôleur gère un domaine spécifique de la page web et est associé à un élément HTML.

javascript/controlleurs/exemple_controller.js

import { Controller } from "stimulus";

export default class extends Controller {
  // Méthode appelée lorsque le contrôleur est connecté à un élément
  connect() {
    // Logique d'initialisation
  }
}

2. Actions

Documentation actions Stimulus

Les actions définissent le comportement d’un contrôleur en réponse à des événements sur ses éléments associés.

index.html

<div data-controller="example">
  <button data-action="click->example#handleClick">Cliquez ici</button>
</div>

javascript/controlleurs/exemple_controller.js

import { Controller } from "stimulus";

export default class extends Controller {
  handleClick(event) {
    // Logique à exécuter lors du clic sur le bouton
  }
}

3. Valeurs de cibles (Target Values)

Documentation targets Stimulus

Les valeurs de cibles sont des éléments HTML associés à un contrôleur qui peuvent être référencés dans le contrôleur.

index.html

<div data-controller="example">
  <input type="text" data-exemple-target="input">
</div>

javascript/controlleurs/exemple_controller.js

import { Controller } from "stimulus";

export default class extends Controller {
  static targets = ["input"];

  // Console log mon input lors du chargement du controller
  connect() {
	console.log(this.inputTarget);
  }

  // Utilisation de la cible
  logInputValue() {
    console.log(this.inputTarget.value);
  }
}

4. Données (Data)

Documentation data-value Stimulus

Stimulus permet de stocker des données dans les éléments HTML à l’aide d’attributs data-*, ce qui facilite la communication entre le HTML et le JavaScript.

index.html

<div data-controller="example" data-controller-name-value="John">
  <!-- HTML -->
</div>

javascript/controlleurs/exemple_controller.js

import { Controller } from "stimulus";

export default class extends Controller {
  static values = {
    name: String // Type de donnée (String, Number...)
  }
  
  connect() {
    console.log(this.nameValue) // => John
  }
}
